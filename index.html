Lista de comandos para hoje!

    Estrutura e como navegar pelo site

  • body: contém tudo que é visual, inclusive vão dentro dele os comandos:
    header: É o cabeçalho do site. footer: É o rodapé do site.
  • ul: O próprio comando de criar lista já um comando novo para vocês: "ul" tudo que está dentro de linhas "li" terá uma bolinha na frente fazendo uma lista.
  • nav: Dentro desse comando você adicionará "links" "a" que se referem à uma parte da sua página (ou a outro site), podendo fazer um sumário do site.
  • Essa linha por exemplo é clicável e vai para a seção que estiver com o id: "exemplo"

    • Dicas de comando

  • Quando precisar usar um comando escreva o nome dele no começo da linha e dê enter, ele colocará toda sua estrutura pronta na linha.
  • No mesmo pensamento ao começar a digitar o código html você pode digitar html:5 e dar enter, isso montará o código base do html para você.
  • Para acompanhar mais rapidamente o seu trabalho, vá nos quatro quadradrinhos ao lado e escreva "live server" e instale essa extensão,
    Assim aparecerá um botão no canto inferior direito do vscode para abrir seu site, e toda vez que você salvar com crtl+S ele atualizará sem precisar recarregar a página.

    • Estilo e texto

  • class: Dentro de cada comando entre as setas comando class:"estilo" sendo o nome depois de class um comando para dar estilo á várias coisas ao mesmo tempo
  • br: em comandos de texto como títulos (h1,h2,h3 por ordem decrescente de tamanho sendo h1 o maior) ou paragráfos (comando p) ou linhas de listas (comando li dentro do ul),
    usar o comando br "quebrará" uma linha dando "enter" para continuar o texto embaixo.

    get embed code, e vá para baixo você verá um link do código que você terá que colocar tanto no HTML quanto no CSS
